Goals:

  • Run the Utility Test Print
  • Adjust the XY Offset

 

 

Running the Utility Test Print

The "Adjust XY Offset" utility has a built-in test print that can be run to see how much, if at all, the XY offsets need to be adjusted by.

To run the utility test print:

  1. Apply glue to your print bed as shown in the picture below and place your print bed into your printer.



  2. Press the menu button in the upper right corner of the printer's LCD touchscreen.



  3. Select "Utilities" from the list of menu options.



  4. Select "Advanced Utilties" from the list of menu options.



  5. Select "Adjust XY Offset" from the list of menu options.



  6. Press the "Print XY Bracket" button.
    Note: The printer will automatically begin printing the part when the button is pressed. Be sure to have leveled and applied glue to the print bed before pressing the "Print XY Bracket" button.

Adjusting the XY Offset

To adjust the XY offset, you will need to know how much the current values are off by. The most accurate way to find this information is to print the test part via the steps above.

To adjust the XY offsets:

  1. Press the menu button in the upper right corner of the printer's LCD touchscreen.



  2. Select "Utilities" from the list of menu options.



  3. Select "Advanced Utilties" from the list of menu options.



  4. Select "Adjust XY Offset" from the list of menu options.

  5. Press the "Adjust" button.



  6. Orient your test part as shown on screen and identify the box on each axis in which the fiber passes through closest to the center.
  7. Use the arrow buttons on the screen to adjust the printer's offset. The lines on screen should intersect those boxes in which the fiber passes through closest to the center on your test part.



  8. When you are sure the offsets are correct, press the "Done" button.
