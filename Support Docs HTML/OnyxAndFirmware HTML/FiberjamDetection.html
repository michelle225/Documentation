Goals:

  • Access the Fiber Jam Detection Utility
  • Determine if Fiber Jam Detection is Enabled
  • Detect a Fiber Jam

 

 

Access the Fiber Jam Detection Utility

To access and enable or disable the Fiber Jam Detection utility:

  1. Select the menu button in the upper right corner of your printer's LCD touchscreen.



  2. Select "Utilities" from the list of menu options.



  3. Select "Advanced Utilities" from the list of menu options.



  4. Select "Fiber Jam Detection" from the list of menu options.



  5. Enable or disable the utility as needed.

Determining if Fiber Jam Detection is Enabled

At any time, even during a print, you can check whether the Fiber Jam Detection utility is enabled. Once a print has started, you will no longer be able to change the status of the Fiber Jam Detection utility without cancelling the print.

To determine if Fiber Jam Detection is enabled on your Mark Two printer:

  1. Select the menu button in the upper right corner of your printer's LCD touch screen.



  2. Select "Settings" from the list of menu options.



  3. Select "Printer Info" from the list of menu settings.



  4. Select the wrench icon and verify the status of the Fiber Jam Detection utility on the screen.

Detecting a Fiber Jam

In the event of a suspected fiber jam, your Mark Two printer will stop the print to allow you to resolve the issue. Once a jam is detected, the part cannot be resumed and will have to be restarted.

If you are unsure about where the fiber is jammed, see No Fiber Coming From Nozzle.

If you suspect that the fiber is jammed in the nozzle, see Replacing the Mark Two Fiber Nozzle.

If you suspect that the fiber is jammed in the extruder, see Fiber Extruder Removal and Reinstallation.
