The Mark One Composite 3D Printer can be updated via the Internet or through a USB drive. Please follow the instructions per the method that suits you.

Update Mark One Firmware via USB

Attention! At the moment, there is currently an issue with updating from a USB stick affecting firmware versions 2015-03-28-20:15:23-f364939 through 2015-05-12-14:01:11-3489f49. This is fixed in the most recent update: 2015-07-22-22:08:46-becf581. If your machine is having problems installing an update from a USB stick you can use the following fix to properly install it: 

  1. Use the Factory Reset USB stick that came with your Mark One to perform a factory reset.
  2. Once the machine has been reset to factory defaults, use another USB stick to install the update as shown below. 

Attention! When installing an update later than November 13, 2015, you may have to install twice via these instructions: Special Guide for Firmware Update 2015-11-13-17:06:52-79b510b

  1. Log into your Eiger.io account and navigate to https://www.eiger.io/settings/about

  2. Download the USB Update

  3. Put the USB update on a USB drive.

    Note: Do not use the MarkForged USB Backup drive that came with your printer. The MarkForged USB drive that came with your printer is a re-flash drive and it will perform a factory reset on your printer.

  4. Once the USB drive is plugged in, you should see a green notification appear at the top of the screen (figure 4.1).

  5. Once the Mark One is done checking the USB drive, a new green notification (figure 5.1) will appear: “Update Available.” You can either press this notification or navigate through the settings page (figure 5.2): “Settings Icon” -> “Settings” -> “Check for Update.”



  6. Once you’re in the Update menu you’ll be presented with the option to apply the update.
    Press “Update” to apply the update (figure 6.1). The update will begin(figure 6.2) and this process will take a few minutes.

    Caution: Do not turn your printer off while the update is in progress. After the update is finished the printer will restart itself.



 

Update Mark One Firmware via WiFi

  1. Make sure your Mark One is connected to the Internet. If so, proceed to step 5.

  2. Connect your Mark One to WiFi by selecting the WiFi icon in the top bar. The Offline icon will be highlighted (figure 2.1).




  3. Make sure WiFi is enabled by selecting the "Wi-Fi"icon(figure 3.1). Select “Configure” to set up your WiFi network connection.



  4. Select your WiFi network from the menu or enter the name of your WiFi network (figure 4.1). If required, enter in your WiFi network password (figure 4.2).

    Note: The Mark One supports WPA and WPA2 network protocols.

    Press “Save” when completed. The WiFi connection may take a minute to authenticate.





  5. From the home screen, please press the settings icon(figure 5.1), then select Settings / Update Manager (figure 5.2).





  6. Once you’re in the Update Manager and if an update is available, the "Could Update" icon will be highlighted in blue. Press “Cloud Update” to apply the update (figure 6.1).

    The update will begin to download (figure 6.2). Once download has completed, the update installation will begin (figure 6.3).

    The update process will take a few minutes.

 

Caution: Do not turn your printer off while the update is in progress. After the update is finished the printer will restart itself.
