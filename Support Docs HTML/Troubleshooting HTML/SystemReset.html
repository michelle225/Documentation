Goals:

  • Create the Factory Reset Key
  • Execute the System Reset
 

Creating the Factory Reset Key  

To create the factory reset key:

  1. Download and unzip the Markforged Factory Reset Image file.
  2. Insert a USB thumb drive into your computer.
    NOTE: any data on the USB Key will be deleted during this process
  3. Download, install and run ImageUSB. ImageUSB is a program that will allow you to turn a regular USB thumb drive into a reset key.
  4. In ImageUSB, select the check box next to the USB drive you wish to reformat.
  5. Select the radio button next to "Write Image to USB" in the "Step 2" panel.
  6. In the "Step 3" panel, browse for the MarkForged Reset Image file that you downloaded earlier.
  7. Click "Write" in the "Step 4" panel. Click "Yes" for all the popups that come up.
  8. Wait for the program to complete and select "OK" when the final popup appears.

Executing the System Reset

To perform a System Reset of your Mark Two printer:

  1. Turn the printer off.
  2. Plug the Recovery USB drive into the USB connector on the back of the printer.
  3. Turn the printer on. The printer’s display will remain dark for the first 30 seconds. After that, commands will appear and scroll down the screen, pausing for long periods of time.
  4. Turn the printer off when the text "System halted" appears at the bottom of the printer display.
  5. Remove the Recovery USB drive.
  6. Wait several seconds and turn the printer back on. This reboot will take longer than subsequent ones. You should be returned to normal default home screen when the boot finishes.
  7. Once your printer’s software has been restored to its factory default state, it is important that you update the software to the latest version available.
