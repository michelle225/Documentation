Access the Print Queue from Eiger

To make printer management easier, you can now save prints in your organization's Print Queue in Eiger. From the print queue page in Eiger you can now print to any printer in your organization.

To access the print queue in Eiger:

  1. Click on the new "Queue" icon in the header.



  2. On the Print page, select "Add to Queue" from the printer dropdown list to add a print to the end of your organization's queue.



  3. Use the "Queue" button from the print queue or print history to requeue a print. Use the "Cancel" button to remove a print from the queue.



  4. Click and drag any print in the queue to move it higher or lower in the queue to print it sooner or later, respectively.



  5. Use the Print button to print from the queue to an available printer. This will remove the print from the queue and start printing it immediately.

Access the Print Queue from a Printer

You can also use the touchscreen on a printer to select a print from your organization's queue and begin printing. Your printer must be connected to the internet to use the Print Queue.

To access the print queue from a printer:

  1. Select "View Queue" on the printer's touch screen.



  2. Navigate through the prints in your organization's queue by using the "Left" and "Right" buttons.



  3. To print a part, ensure it is selected and press the "Print" button.
