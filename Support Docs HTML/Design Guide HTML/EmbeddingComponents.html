Embedding Components Inside a Part

Electronics and other components can be embedded inside your part.  

To embed a component inside a part during a print:

    Before Printing:
      1. Model a cavity for the component and export the STL file.
      2. Open Eiger and import the STL file with the cavity
      3. Verify the print settings and save the part
      4. Click on "Internal View" near the bottom right of the screen.
      5. Click on "2D" near the top right of the screen.
      6. Use the layer bar at the bottom of the screen to select a layer to pause the print at.
        Note: The selected layer should be one in which the cavity is still open but is also deep enough to hold the component without causing a collision with the print head.
      7. With the layer selected, toggle the "Pause After Layer" button under the "Editing Layer" heading at the top right of the screen.

    During the Print Process:
      1. Start your print and wait for the printer to pause after the layer indicated.
      2. Use the supplied glue stick to apply glue to top and bottom of the component that will be embedded.
      3. Place the component into the cavity and ensure that it is firmly in pace.
      4. Ensure the component is below the print nozzle, when placed in the cavity, or the print nozzle may collide with the part and be damaged.
